# 會員註冊表單

## 專案概述
- 此專案整合了 DOM 事件委派、Constraint Validation API、客製化中文錯誤訊息。
- 主要功能包括 : **即時驗證**、**送出攔截**和**防重送機制**。



## 核心功能

1. **即時驗證**：   
欄位在 `blur` 後啟用錯誤提示，`input` 時即時更新。

2. **客製化訊息**：  
用 `setCustomValidity()` 搭配 `<p>` 元素，顯示客製化的中文錯誤說明。

3. **送出攔截與防重送**：
    - `submit` 事件會檢查所有欄位有效性。
    - 失敗時，會自動**聚焦**到第一個錯誤欄位。
    - 成功時，按鈕會變為 `disabled` ，防止重複送出。

4. **事件委派**：  
**興趣標籤**區塊使用父層監聽切換選中狀態。

5. **可及性設計**：  
所有錯誤提示均透過 `aria-describedby` 與欄位連結。

6. **重設功能**：  
清除所有欄位內容、錯誤訊息，並重設興趣標籤的選中狀態。



## 欄位與自訂錯誤條件

| 欄位名稱 | 驗證條件 | 錯誤訊息 |
| :--- | :--- | :--- |
| **姓名** | 必填 | `此欄位為必填。` |
| **Email** | 必填，驗證相符格式 | "請輸入有效的電子郵件格式。" |
| **手機** | 必填，限制 10 碼數字 | "請輸入 10 碼數字的手機號碼。" |
| **密碼** | 必填，至少 8 碼，英數混合 | "密碼長度至少需要 8 碼。", "密碼必須包含英文字母和數字。" |
| **確認密碼** | 必填，需與密碼一致 | "確認密碼與密碼不一致。" |
| **興趣標籤** | 必填，至少點選 1 個 | "請至少選擇 1 個興趣標籤。" |
| **服務條款** | 必填（Checkbox 需勾選） | "此欄位為必填。" |

## 進階功能

- **密碼強度指示條**：  
實作了強度判斷邏輯，並即時更新視覺回饋，弱 (紅色) / 中 (橘色) / 強 (綠色)。

- **「重設」按鈕** :   
清除所有欄位與錯誤訊息，並重置強度條和興趣標籤。
